# VentusALL
# A empresa “VentusALL” é uma loja virtual que vende jogos digitais para várias plata formas 
(como PC, PlayStation e Xbox), com o público-alvo sendo jovens entre 16 a 25 anos. O propó
sito é trazer um meio rápido e prático de comprar jogos online para qual quer plataforma. O 
sistema deve cadastrar clientes, gerenciar a venda de jogos (inclu indo transações com múlti
plos itens) e registrar vendas com valor total, data e chave de ativação individualizada por 
jogo.
